    Introducing Apache Wicket

    Apache Wicket is a Java based, open source, component oriented web application framework. Instead of writing request/response handlers you create reusable components using HTML and Java.

    Plain HTML markup

    Using plain HTML markup—extended with specific Wicket markers—you create rich web pages without having to learn a new tag language. Use the language of your web designers.

    <html>
<head>
    <title>Hello, World!</title>
</head>
<body>
    <h1 wicket:id=”text”>replace me</h1>
</body>
</html>

    Plain Java coding

    Coding is done in plain old Java. No XML programming, no annotation hunting.

    package com.mycompany.app;

import org.apache.wicket.markup.html.WebPage;
import org.apache.wicket.markup.html.basic.Label;

public class HelloWorld extends WebPage {
   public HelloWorld() {
       add(new Label("message", "Hello World!"));
   }
}
